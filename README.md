# Delícias Caseiras — Bakery Landing / E‑commerce

Demo: [https://delicias-caseiras-ba191.web.app](https://delicias-caseiras-ba191.web.app)

---

## Project Overview

Delícias Caseiras is a simple, responsive landing page / light e-commerce for a bakery. It’s built to showcase products, collect orders via EmailJS, and present customer testimonials. The design includes a mobile-friendly hamburger menu and focuses on accessibility and responsiveness.

---

## Features

* Clean, bakery-focused landing page layout
* Product showcase / menu sections
* Customer testimonials section
* Order form powered by EmailJS
* Responsive design (desktop, tablet, mobile)
* Hamburger menu for small screens

---

## Quick Start

1. Clone the repository:

```bash
git clone https://github.com/josueeliezer01/delicias-caseiras.git
cd delicias-caseiras
```

2. Install dependencies:

```bash
npm install
```

3. Run the development server:

```bash
npm start
```

4. Build for production and deploy to your static host of choice:

```bash
npm run build
```
